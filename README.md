# Wine Classification & Data Exploration

This project explores the UCI Wine dataset (https://www.kaggle.com/datasets/tawfikelmetwally/wine-dataset) and applies several machine learning models to classify wines based on chemical properties.

## Dataset
The dataset contains chemical analysis results of wines derived from three different cultivars grown in the same region in Italy.

Features include:
- Alcohol
- Malic Acid
- Flavanoids
- Color Intensity
- Proline
- etc.

## Project Workflow

1. Exploratory Data Analysis
2. Feature Scaling
3. Model Training
4. Model Comparison
5. PCA Visualization

## Models Used

- Logistic Regression
- Random Forest
- K-Nearest Neighbors
- Support Vector Machine
- Decision Tree
- Naive Bayes

## Results

Best performing model: **Random Forest / Logistic Regression** with 100% accuracy.
